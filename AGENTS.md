# WoofTennis — Agents

Контекст проекта автоматически загружается через `.cursor/rules/` — читать дополнительные файлы перед работой не нужно.

## Агенты

| Файл | Когда использовать |
|---|---|
| `.agents/FE.md` | Изменения в `apps/web/` — страницы, компоненты, стили |
| `.agents/BE.md` | Изменения в `apps/api/` — эндпоинты, сервисы, миграции |
| `.agents/REVIEW.md` | Ревью готовых изменений перед мержем |

## Как ставить задачу

Открой нужный агентный файл в Cursor, добавь задачу в конце:

```
ЗАДАЧА: <конкретное описание>
ФАЙЛЫ: <какие файлы затронуты или нужно создать>
НЕ ТРОГАТЬ: <что явно исключено из скоупа>
```

## Правила для всех агентов

- Читать только файлы, релевантные задаче — не делать обзорный обход всего репо
- Не менять несвязанные участки кода
- При изменении типов в `packages/shared` — пересобрать: `npm run build --workspace=@wooftennis/shared`
- Проверки перед сдачей: `npm run lint` + `npm run build` для затронутого workspace

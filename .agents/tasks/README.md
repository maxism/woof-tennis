# Tasks Folder

Эта папка хранит задания ролям в стандартизированном Markdown-формате.

Перед созданием новой задачи:
1. Прочитать `.agents/TASKS-FORMAT.md`
2. Создать новый файл по шаблону именования
3. Передать содержимое файла соответствующей роли

Рекомендуется один файл = одна задача = одна роль.
